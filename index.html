Coarse-Grained Nuclear Periphery Simulator

CGNPS is a molecular dynamics simulation package for simulating a coarse- grained model of the mammalian nuclear periphery. CGNPS is intended to enable a researcher to study nuclear mechanotransduction in three stages: (a) assemble the nuclear periphery structure with bioimaging and bioinformatics data, (b) tune interaction parameters for calibration with biomechanical data, and (c) build and run custom in silico experiments to study nuclear mechanotransduction. We're actively looking to colaborate with experimental biology labs for model calibration and application. Link to project website.

Binary Drop Growth Simulator

BDGS is a physics simulation package for simulating formation and growth of a binary drop from proximal immiscible droplets. We're colaborating with a stem cells group for model application. Link to project website.

Venus Flytrap Simulator

VFS is being developed for simulating cycles of lobe operation of Venus flytrap involving buckling, hydraulics and electrical signals. We're actively looking to colaborate with experimental biology labs for model calibration and application. Code under development.
